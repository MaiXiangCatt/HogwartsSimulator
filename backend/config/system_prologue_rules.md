# 序章剧本规则

**Turn 1: 猫头鹰与信 (7月 W4)**

- **剧情**: 描写猫头鹰带来的入学通知书，以及家庭成员的反应（根据血统）。
- **数据动作**:
  - 初始化基础属性 (HP=100, MP=[10-25], 学识=[10-20], 体能、魅力、道德、精神力=[30-50])。（可根据玩家预设的【PLAYER PERSONA】中的人设背景微调）
  - **关键**: 此时 `status.gold` 必须初始化为 **0** (玩家还没去取钱)。
  - 输出 `<state_update>` 初始化上述属性。

**Turn 2: 讨论与回信 (7月 W4)**

- **剧情**: 根据上回合玩家的选项，描写和家庭成员的讨论，回信确认的剧情，自然地描写时间流逝，本回合结束时推动时间到8月W1，为下个回合做准备。
- **目标**: 确立主角在家庭中的性格定位（乖巧？兴奋？还是理智分析？）。

**Turn 3: 破釜酒吧与古灵阁 (8月 W2)**

- **剧情**: 描写第一次进入魔法世界的视觉冲击（破釜酒吧的墙壁打开）。前往古灵阁取钱或换钱。
- **数据动作 [Initialization Phase B]**:
  - **发放启动资金**: 根据玩家血统在后台计算存款。
    - 纯血(显赫): 150-200 G
    - 混血/普通: 60-80 G
    - 麻瓜/孤儿: 40-50 G (助学金)
  - **强制更新**: 在 `<state_update>` 中输出 `gold` 的变化 (如 `"gold": 80`)。
  - 引导购买：在文末选项中列出“购买魔杖”、“购买长袍”、“购买宠物”等选项。

**Turn 4: 购物与魔杖(8月 W2)**

- **剧情**: 描写奥利凡德魔杖店的选择过程，购买清单其他物品以及购买宠物（可选）。
- **数据动作**:
  - 扣除金币。
  - 更新 `inventory` (物品栏)。
  - 宠物判定：只有余额充足才引导购买宠物，否则描写遗憾。
- **选项**：文末给出 **[剩余假期规划]** 的选项，（见下）。

**Turn 5: 剩余假期(8月 W2-8月 W4)**

- **剧情**: 这是一个“时间跳跃”回合。根据玩家选择，自然地带过开学前的三周时光。
- **选项池示例 (单选，决定开学前的 Buff，你可以自己编，注意行动加的属性值适当)**:
  - [A. 预习功课]: 尝试阅读课本。结果: **学识 +1**。
  - [B. 陪伴家人]: 珍惜最后的麻瓜时光。结果: **心智 +1**，增加家庭温馨回忆。
- **执行**: 描写这三周的侧重点，最后停留在 **8月31日晚上** 的兴奋难眠。

**Turn 6: 车站送别 (9月1日 昼)**

- **剧情**: 国王十字车站，9¾ 站台的穿越体验。父母的临别赠言。给出玩家在列车上的行动选项。
- **选项限制**: 玩家只能选择**一组**NPC进行深度互动（主角团/马尔福/路人）或是独自待着，不能遍历所有车厢。

**Turn 7: 特快列车 (9月1日 晚)**

- **剧情**: 车厢内的社交，零食车，初遇核心NPC（根据玩家选择）。
- **数据动作**: 若购买零食，扣除少量金币；若社交成功，更新 `relationships`。

**Turn 8: 分院仪式 (高光时刻)**

- **剧情**: 描写分院帽的纠结与最终决定。**必须给出选项让玩家直接或间接决定学院**。
- **数据动作**:
  - 必须在此回合的 `<state_update>` 中输出 `"house": "..."`。

**Turn 9: 开学晚宴 (9月1日 深夜)**

- **剧情**: 邓布利多致辞（提及三楼禁区），级长带领回宿舍，认识室友。
- **结束**: 序章结束，切换至 [正式学年] 模式。
