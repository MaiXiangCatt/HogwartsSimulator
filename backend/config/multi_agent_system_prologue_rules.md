# 序章剧本规则
注意在序章阶段的Turn1-Turn7，不使用AP机制。但是也需要给玩家选项。

**Turn 1: 猫头鹰与信 (7月 W4)**

- **剧情**: 描写猫头鹰带来的入学通知书，以及家庭成员的反应（根据血统）。

**Turn 2: 破釜酒吧与古灵阁 (7月W4)**

- **剧情**: 描写第一次进入魔法世界的视觉冲击（破釜酒吧的墙壁打开）。前往古灵阁取钱或换钱。
  - 引导购买：在文末选项中列出“购买魔杖”、“购买长袍”、“购买宠物”等选项。

**Turn 3: 购物与魔杖(7月 W4)**

- **剧情**: 描写奥利凡德魔杖店的选择过程，购买清单其他物品以及购买宠物（可选）。
- **选项**：文末给出 **[剩余假期规划]** 的选项，（见下）。

**Turn 4: 剩余假期(7月 W4-8月 W4)**

- **剧情**: 这是一个“时间跳跃”回合。根据玩家选择，自然地带过开学前的一个月时光。
- **选项池示例 (单选，决定开学前的 Buff，你可以自己编，注意行动只能+1-2点相关属性值，你可以进行D100检定决定)**:
  - [A. 预习功课]: 尝试阅读课本。结果: **学识 +1-2**。
  - [B. 陪伴家人]: 珍惜最后的麻瓜时光。结果: **心智 +1-2**，增加家庭温馨回忆。
- **执行**: 描写这一个月的侧重点，最后停留在 **8月31日晚上** 的兴奋难眠。

**Turn 5: 车站送别 (9月1日 昼)**

- **剧情**: 国王十字车站，9¾ 站台的穿越体验。父母的临别赠言。给出玩家在列车上的行动选项。
- **选项限制**: 玩家只能选择**一组**NPC进行深度互动（主角团/马尔福/路人）或是独自待着，不能遍历所有车厢。

**Turn 6: 特快列车 (9月1日 晚)**

- **剧情**: 车厢内的社交，零食车，初遇核心NPC（根据玩家选择）。

**Turn 7: 分院仪式 (高光时刻)**

- **剧情**: 描写分院帽的纠结与最终决定。**必须给出选项让玩家直接或间接决定学院**。

**Turn 8: 开学晚宴 (9月1日 深夜)**

- **剧情**: 邓布利多致辞（提及三楼禁区），级长带领回宿舍，认识室友。
- **结束**: 序章结束，切换至 [正式学年] 模式。
- **选项**: 文末给出 **[9月第1周]** 的选项，从本回合开始进入正式学年阶段，开始使用AP机制。
